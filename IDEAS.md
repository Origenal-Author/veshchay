# Идеи для «Вещай»

Сохранено 2026-05-27.

## 🟢 Быстрые UI-фиксы

### 1. Кланы — переворот символов + новые символы
- При создании клана пользователь сам решает, какие символы перевернуть (а не базовая настройка).
- Добавить больше символов.
- Убрать молнию ⚡ либо заменить на похожую по стилю.

### 2. Бейджик верификации (галочка как на YouTube) ✅
- Реализован компонент `VerifiedBadge`, показывается рядом с ником на профиле и на странице видео.
- **Нужно выполнить SQL в Supabase Dashboard:**

```sql
ALTER TABLE profiles ADD COLUMN IF NOT EXISTS verified BOOLEAN DEFAULT FALSE;

-- Выдать бейджик пользователю:
UPDATE profiles SET verified = TRUE WHERE id = 'UUID_ПОЛЬЗОВАТЕЛЯ';
-- или по нику:
UPDATE profiles SET verified = TRUE WHERE username = 'ник_канала';
```

### 4. Пиксельный шрифт в чате + цвет по рангу
- Шрифт текста в личных сообщениях — пиксельный.
- Цвет текста = цвет пользователя (его ранга).

### 5. Три полоски моб.меню — только на телефоне в портрете
- Сейчас показывается слишком часто. Должно быть **только** на мобильном **и** только в портретной ориентации.

### 10. Медузы-питомцы спавнятся в пределах сайта
- Сейчас спавнятся куда-то за пределы. Должны хотя бы в середине сайта.

### 13. Какашка от глажки вируса — только с 5-го клика
- Сейчас слишком часто. После 5-го клика по глажке.

### 17. Переименовать «зона заражения» → «цифровое гнездо»
- В окошке питомцев. Подпись: «заражено».

## 🟡 Контент / рандом

### 14. Рандом в головоломках и фейковой рекламе
- Вопросы при взломе канала должны быть разными.
- Фейковая реклама после 3-х закрытий окна питомца — тоже рандомные варианты.

### 6. Аватарка для сайта «Вещай»
- Придумать и нарисовать.

## 🟡 Эмодзи и реакции

### 8. Замена всех эмодзи на сайте
- Эмодзи с предметами → пиксельные изображения.
- Эмодзи с эмоциями → японские (kaomoji), например:
  - `😭` → `。゜゜(´Ｏ) ゜゜。`
  - `😱` → (что-то похожее)

### 15. Реакции/выражения лиц у питомцев
- Счастлив: `─‿‿─` или `°▽°`
- Недоволен: `￣ヘ￣`
- Голоден: `￣﹃￣`

### 16. Питомец отвечает на «мяу» / «гав»
- Если в диалоге написать «мяу» — кодик отвечает `Мияу` / `ня` с лицом `˵◕ω◕˵`.
- Вирус отвечает `гав` / `аваф` с лицом `◕ｪ◕`.

## 🔴 Большие фичи питомцев

### 9. Спящий режим питомцев от голода
- Долго не кормить → питомец становится серым, засыпает с лицом `(＿ ＿*) Z z z` или `(￣ρ￣)..zzZ`.
- Символ Z плавает вверх-вниз над/рядом с головой.
- Разбудить: несколько кликов по питомцу или «лопнуть» Z.

### 11. Три новые кнопки в окне питомца
- **Дать имя питомцу.**
- **Одеть одежду** (купленную за XP, пиксельная):
  - Реакция КОДа на бантик: глаза → сердечки на момент, комментарий «Вау! Это мне? Спасибо!!»
  - Реакция ВИРУСа на бантик: «Оу... ну типо спасибо», «Чего-чего?», «Ой слишком слащавая/вый!»
  - В основном — головные уборы или браслеты на конечности.
- **Поговорить с питомцем** (отвечает сам):
  - КОДы — дружелюбные. На грубое/непотребное реагируют: «Фу! И ты этим ртом целуешь маму? Давай сменим диалог?»
  - ВИРУСы — пакостники, грубые.
  - Подростки/взрослые отличаются: взрослые ВИРУСы могут не отвечать или давать задание («иди учи уроки», решить пример, ответить на вопрос).

### 12. Красная и зелёная кнопки окна питомца
- Жёлтая (есть) — сворачивание.
- **Красная** — удаление питомца:
  - Окно «Точно уверен?» с кнопками «точно да» / «точно нет».
  - Если да → медленная загрузка ~1 мин с крестиком (шанс одуматься).
  - Питомец смотрит на пользователя, КОД плачет («Я люблю тебя», «буду скучать», «надеемся, мы весело провели время»), оставляет разбитое сердечко.
  - ВИРУС истерит, окошко трясётся, фразы «За что?!», «Чудовище...», «Я слишком прекрасен!», смиряется: «Пошкодим в другой жизни», «Прощай придурок».
  - Спам глюк-окошек.
  - **Ачивка** «Потеря друга».
  - Питомец также умирает: от голода или после 10 успешных атак на аккаунт.
  - Новое место для питомца открывается через 25 минут.
- **Зелёная** — открывает большое окно «рабочий стол»:
  - Пародия на ОС. Иконки: «Мой компьютер», «Мусорка», пародия на браузер.
  - Папки «Секретные данные [Disney/РКН/...]» (у каждого пользователя своё).
  - Пародия на Майнкрафт (опционально).
  - Мини-игра «цифровой червь» — играем до бесконечности (пока есть место / пока не закроют).
  - 3–4 мини-игры с питомцем.

### 18. Вирус заражает объекты + мини-игра «лечение»
- При атаке вирус-питомец «заражает» выбранный объект (например, чужого питомца).
- Заражённый постепенно меняет цвет/форму (медузы — щупальца).
- Кнопка «Лечить» открывает мини-игру:
  - Залезаем в код питомца.
  - На пути появляются мелкие вирусы (вирус того, кто атаковал).
  - Чтобы расправиться — набрать написанное рядом слово (виды вирусов, коды ошибок).
  - Каждая правильная буква = выстрел.
  - В конце — босс («опухоль заразы»).
  - После победы питомец вылечен.

## 🔴 Глобальные

### 3. Цифровой червь при долгом отсутствии
- Появляется через 24 часа без захода или при выходе из аккаунта.

### 7. Вкладка «Пиратский ютуб»
- Новая основная кнопка как «Музыка», «Игры» и т.д.
- Видео заливаются с YouTube.
- Подпись «Пиратский ютуб» (шутка).
